# Personal Finances Frontend

Vue 3 + Nuxt 4 frontend for the Personal Finances app.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Opens at <http://localhost:3000>

## Production

```bash
npm run build
npm run preview
```

## Linting

```bash
npm run lint
npm run lint:fix
```
